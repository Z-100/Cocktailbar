Create and View individual cocktails and their corresponding recipes!

## Requirements

* As a member
  * Login
  * Logout
  * Manage account
  * Delete account
  * Upload new CR
  * View CR
    * Search specific CR
    * Get recommended CR
      * (Dashboard)
  
* As a visitor
  * Register
  * View CR
    * Search specific CR
    * Get recommended CR
